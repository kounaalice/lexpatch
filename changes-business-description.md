# LexCard 事業説明の修正仕様

## 方針

LexCardの立ち位置を「営利企業が公共目的で無償提供する法令アクセス基盤」に統一する。
料金ページは非表示にせず、法令アクセス支援の箇所に「公共目的のため無料で提供」と明示する。

---

## 1. 料金ページの修正（非表示ではなく内容修正）

### `/lp` ランディングページ

現在の料金プラン表示（Free/Standard ¥240,000/Enterprise ¥900,000）の
法令アクセス支援（LexCard）に関する部分に以下を明示:

```
法令アクセス支援（LexCard）は、公共目的のため無料で提供しています。
```

有料プラン（Standard/Enterprise等）は、コード・システムの移植による
カスタマイズ別注として位置づけを明確にする。

冒頭に以下を追加:

```
LexCardの法令検索・改正提案・コミュニティ機能は、
公共目的のためすべて無料で提供しています。

以下の組織向けプランは、コード・システムを移植しての
専用環境構築（カスタマイズ別注）をご検討の方向けです。
```

```
of the LexCard system in dedicated environments.
```

## 6. 利用規約の修正検討

以下の趣旨を利用規約に反映:

- サービスの利用は無料
- ソースコード・システムの複製・転用は禁止
- APIの無断大量利用・再販は禁止
- LexCardのデータ・UIを使った類似サービスの構築は禁止
- カスタマイズ（コード・システムの移植）は別注にて対応

## 7. 注意事項

- 「非営利」という表現は使わない（こうな姫株式会社は営利法人）
- 「公共目的で無償提供」「収益化を目的としない」が正確な表現
- 営利企業であることは隠さず、むしろ強みとして言及する
