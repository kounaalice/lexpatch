# 自治体法務の現状と課題

## 法令情報へのアクセス

| 現状 | 課題 |
|------|------|
| 紙の六法・加除式法規集 | 改正反映が遅い、検索できない、持ち運べない |
| e-Gov法令検索 | UIが使いにくい、チームで共有できない |
| 商用法令DB（Westlaw/D1-Law等） | 年額60〜300万円、入札が必要、ID数で課金 |

## 条例改正業務の実態

- 新旧対照表はWordで手作業 → **数日かかる**
- 改め文は経験者頼み → **属人化**
- 法制執務のナレッジが共有されない → **異動で消失**
- 庁外（出張先・在宅）からアクセスできない → **VPN/LGWAN依存**

# 主要機能

## 1. 法令閲覧
- e-Gov法令APIから取得した**最新法令テキスト**をカード形式で表示
- 分野別カテゴリ（50分野）、全文検索、条文ジャンプ

## 2. 改正案作成（パッチ）
- 条文を**直接編集**して改正案を作成
- **新旧対照表**（差分ビュー）を自動生成、**改め文**をワンクリック生成

## 3. 逐条解説
- 条文ごとにコメンタリーを付与、組織内のナレッジとして蓄積
- **担当者の異動後もナレッジが残る**

## 4. プロジェクト管理・法令フォロー
- 条例改正案を「プロジェクト」として管理、関連法令の改正をウォッチ
